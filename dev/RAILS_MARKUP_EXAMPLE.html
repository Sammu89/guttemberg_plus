Class Name Convention Summary

Element Class Name Purpose
Parent Block wp-block-guttembergplus-rails Main wrapper (WordPress naming convention)
Theme Class rails-theme-{name} Applied theme name (e.g., rails-theme-modern)
Rail Container rails-rail Container for items (left or right)
Left Rail rails-rail-left Left-side rail modifier
Right Rail rails-rail-right Right-side rail modifier
Content Stage rails-stage Panel display area (center)
Rail Item (Button) rails-item Individual button in rail
Item Content Wrapper rails-item-content Flex container for numeral/label/icon
Item Numeral rails-item-numeral Number/numeral text (01, 02, etc.)
Item Label rails-item-label Button label text
Item Icon rails-item-icon Icon element (▾, ▴, etc.)
Content Panel rails-panel Content area for active item
Active Panel rails-panel-active Currently visible panel modifier
Scroll Button rails-scroll-button Up/down scroll button (overflow mode)
Scroll Up rails-scroll-up Up button modifier
Scroll Down rails-scroll-down Down button modifier
Tabs Row (Mobile) rails-tabs-row Horizontal tab row on mobile
Accordion Container rails-accordion-container Accordion wrapper on mobile
Accordion Item rails-accordion-item Individual accordion item
Accordion Button rails-accordion-button Accordion toggle button
Accordion Panel rails-accordion-panel Accordion content area

Data Attribute Convention Summary

Attribute Element Values Purpose
data-layout-mode Parent block nextToButton | corner Panel positioning mode
data-height-mode Parent block matchTallest | maxHeight Height calculation strategy
data-max-height Parent block CSS length (600px, 80vh) Maximum panel height
data-overflow-behavior Parent block scrollbar | buttons Overflow handling method
data-responsive-mode Parent block desktop | mobile Current responsive state
data-mobile-layout Parent block tabs | accordion | none Mobile transformation type
data-responsive-breakpoint Parent block Number (768) Breakpoint in pixels
data-compression-ratio Parent block 0-1 (0.3) Button compression amount
data-compression-duration Parent block Milliseconds (300) Animation duration
data-rail-side Rail container, Item left | right Which rail this belongs to
data-expansion-direction Item left | right Panel expansion direction
data-tilt Item -90 | +90 Vertical text rotation angle
data-item-index Item Number (0, 1, 2...) Item position in sequence
data-direction Panel left | right Panel slide animation direction
data-transition Panel slide | fade | none Panel animation type
data-overflow Panel scrollbar | buttons Active overflow mode
